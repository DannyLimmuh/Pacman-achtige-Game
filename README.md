# Pacman-achtige-Game

##-- Game idee
Vrij logisch, een pacman achtige game  alle bolletjes moet pakken voor de volgende level

##-- Doel van de speler
Je moet alle bolletjes pakken voor de volgende level

##-- Wat doet de speler
Vijanden ontwijken en bolletjes pakken om naar de volgende level te gaan (mogelijkheid om vijanden te verslaan bij oppakken van bepaalde item)

##-- Twist
Portal gun gimmick
